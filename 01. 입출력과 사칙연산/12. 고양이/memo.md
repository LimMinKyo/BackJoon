백틱을 사용하면 여러 줄을 한 번에 출력할 수 있다.

## 백슬래시 사용하기

- \n : 줄 바꿈 (Enter)
- \\ : 백 슬래시
- \' : 작은 따옴표
- \" : 큰 따옴표

## 답안
```js
// 백틱(`)이용
  const cat =
  `\\    /\\
   )  ( ')
  (  /  )
  \\(__)|`;
  console.log(cat);
```
```js
// \n와 문자열 연산 이용
  const cat =
  "\\    /\\\n"+
  " )  ( ')\n"+
  "(  /  )\n"+
  " \\(__)|";
  console.log(cat);
```
```js
// \n과 역슬래시\ 이용
  const cat =
  "\\    /\\\n\
   )  ( ')\n\
  (  /  )\n\
  \\(__)|";
  console.log(cat);
```
